* **자료형**

  1. **숫자형 - 숫자**

     표현: `정수` `실수(소수점)` `8진수(0o)` `16진수(0x)` 

     연산: `사칙연산(+, -, *, /)` `제곱(**)` `나머지(%)` `몫(//)`

     

  2. **문자열 - 문자**

     표현: 따옴표 안에 써 넣음

     종류: `큰 따옴표(" ")`, `작은 따옴표(' ')`

     *인용을 위해 혼용한다

     줄바꿈 허용: `큰 따옴표(""" """)`, `작은 따옴표(''' ''')`

     

     **Escape code**

     `\n` `\t` `\\` `\'` `\"` `\r` `\a` `\b` `\000`

     

     **연산**: 

     `더하기(+)` concatenation - 문자열

     `곱하기(*)` *숫자 - 같은 문자열 반복

     `len()` 문자열의 길이(문자 개수)를 세어 줌 *공백 포함

     

     **포멧팅**: 문자열의 형식을 만들어주는 작업

     * 대입: 

       숫자 대입 - 'I eat %d apples' %3

       문자열 대입 - 'I eat %s apples' %thress *숫자 넣으면 자동변환

       변수 대입 - number = 3 

       ​				'I eat  %d apples' %number

       2개 이상의 값 넣기 - 'I eat %s apples for %d days' %(number, 3)

     * 자릿수 설정: 

       표현: %(공백)자릿수.소수점(자료형) %입력	

       EX: %-10.4s $3.141592 - '    3.1416'

       ​      오른쪽 공백, 10자리, 소수점 4자리까지 반올림해서 표시

       

     * 포멧 코드와 숫자 함께 이용하기 by 함수

       * 대입: 

         기본: '문자열'.format('문자열'/숫자, '문자열'/숫자, ...)

         변수 대입: ` 'I ate {n} apples at {day} day'.format(n = 3, day = 'first')` 

         인덱스 이용: ` 'I ate {0} apples at {day} day'.format(3, day = 'first')` 

       * 정렬 및 공백:

         표현: '{0:(공백을 채울 문자)(정렬)(자릿수.소수점 자리)}'.format('문자열'/숫자)

         EX: ` '{0:=^10.4f}'.format('hi')` 

         

     * **f-formating**: f 문자열 포멧팅 *작은 따옴표(' ')만 이용하니까 주의

       * 대입: 

         변수 설정하고 {변수}로 넣어주면 됨

         ```
         name = '정영우'
         age = 26
         f '이름은 {name}, 나이는 {age + 1}'  
         ```

         딕셔너리 활용

         ```
         d = {name: '정영우', age: 26}
         f '이름은 {d['name']}, 나이는 {d['age']}'  
         ```

         - 정렬 및 공백 채우기: 

         - 표현: f'{(문자열):(공백)(정렬방식)(자리수)}

           EX: `f’{“hi”:!<10}`

           

     * **문자열 함수**: **입력값**을 넣으면 특정한 **출력값**을 내는 것

       `count()` `find()` `index()` `join()` `upper()` `lower()` `lstrip()` `rstrip()` `strip()` `replace()` `split()` ...

     

  3. **리스트**

     표현: [1, 2, 3, 4]

     만들기: 

     `<변수> = [<요소1>, <요소2>, <요소3>, ...]`

     `<변수> = list(문자열, tuple)`

     `<변수> = []` 

     

     **연산**

     `더하기(+)` 

     `곱하기(*)` *숫자 - 리스트 내 변수 반복

     

     ***인덱싱*** - 특정 위치에 있는 리스트 변수를 지명

     표현: 

     `list명[인덱스 번호]` *변수 위치는 0부터 시작

     `list명[1차원 인덱스 번호][2차원 인덱스 번호]`

     EX: a[0] - 리스트 a 안의 첫번째 변수 *음수는 뒤에서 부터

     

     **슬라이싱** 

     개념: 변수를 범위에 맞추어 자름

     EX: 리스트명 [변수 위치:변수 위치] *변수 위치 디폴트 - 처음:끝

     예시: a[:3] - 리스트 a 안의 변수 중 0번부터 2번까지

     

     **리스트 함수**: **입력값**을 넣으면 특정한 **출력값**을 내는 것

     표현: `list명.함수(입력값)`

     종류: `len()` `append()` `sort()` `reverse()` `index()` `insert()` `remove()` `pop()` `extend()` ...

     

  4. **튜플**

     만들기:

     특징: 수정 불가 

     인덱싱, 슬라이싱:

     `tuple명[인덱스 번호]`

     연산: 

     `더하기(+)` 

     `곱하기(*)` *숫자 - tuple 내 변수 반복

     

  5. **딕셔너리**

     **표현**: `{key1: value1, key2: value2, key3: value, ...}`

     **만들기**: 

     `<변수> = {}`

     `<변수> = {key1: value1, key2: value2, key3: value, ...}`

     **특징**:

     key는 고유한 값으로 중복이 불가함, 따라서 key로 문자열과 tuple은 가능하지만 (수정이 가능한) list는 불가능  

     **딕셔너리**

     * 수정

       요소 추가하기: `dictionaty명['key'] = 'value'`

       요소 삭제하기: `del dictionaty명['key']`

       비우기: `dictionary명.clear`

     * 요소 나열 *list나 tuple로 변환해서 이용 가능

       key 나열: `dictionary명.keys()` *type() > dict_keys

       value 나열: `dictionary명.values()` *type() > dict.values()

       전부 나열: `dictionary명.items()` *type() > dict.items()

     * 추출

       key로 value 추출: `dictionary명.get('key')`

       key가 dictionary 안에 존재하는지 확인: `key in dictionary명` > True or False

       

  6. **집합**

     표현: `{요소1, 요소2, 요소3, ...}`

     만들기****: 

     `<변수> = {}`

     `<변수> = set(문자열, list, tuple)`

     특징:

     1. 중복값을 허용하지 않음 *중복 제거를 위해 집합을 이용하기도 함
     2. 인덱스 번호가 없음 *요소를 확인/수정하기 위해서 list나 tuple로 변환하기도 함

     연산:

     * 집합-집합

       교집합: `집합1 & 집합2`

       합집합: `집합1 | 집합2`

       차집합: `집합1 - 집합2`

     * 집합-원소

       단일 원소 추가: `집합1.add(원소)` 

       집합 추가: `집합1.add(집합2)` 

       단일 원소 소거: `집합1.remove(원소)` *원소가 집합1 내에 없으면 Error 발생

       단일 원소 소거: `집합1.discard(원소)` *원소가 집합1 내에 없어도 정상종료
